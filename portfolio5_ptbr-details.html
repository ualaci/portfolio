Detalhes

Trabalho de conclusão de curso focado na aplicação de técnicas de mineração de dados para analisar o banco de dados da Síndrome Respiratória Aguda Grave (SRAG), fornecido pelo Ministério da Saúde do Brasil. Foram utilizados algoritmos de aprendizado de máquina, incluindo árvores de decisão, FP-Growth e redes neurais, para identificar padrões clínicos e fatores preditivos. Os principais resultados indicaram que o diagnóstico da doença, a necessidade de ventilação mecânica e a idade foram preditores críticos dos resultados clínicos. O estudo também desenvolveu uma plataforma interativa de visualização de dados para permitir a análise detalhada das relações entre as variáveis. Apesar das limitações, como a exclusividade dos dados ao Brasil, a pesquisa oferece insights para melhorar a gestão da saúde pública e a resposta a surtos.
